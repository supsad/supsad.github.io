План работы:
1) Веб-райтер (или свободный член команды) — добавляет контент к сайту с помощью любых источников информации.

2) Кодер — с помощью JavaScript создает различные механизмы тестов от стандартных (выбрать из нескольких вариантов ответа) и самостоятельного ввода ответа, до проверки ошибочного решения (кликнуть на ошибку в решении). Также руководит проектом.

3) Дизайнер — с помощью CSS и HTML оформляет сайт в удобном для чтения виде. Также может добавить фреймы: Прогноз погоды, Цитаты, Анекдоты и т.д.
