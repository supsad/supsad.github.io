SlfStudy. - сайт самообучения.

Сайт сделан на основе html, javascript и css, а также плагинов-расширителей sass, gulp.

Для чего он нужен?

Он поможет студентам подготовиться к сдачей экзаменов и зачетов.

На сайте есть выбор дисциплины для самоподготовки, темы, а также выбор между теорией и практикой (последнее реализовано в виде тестов).
